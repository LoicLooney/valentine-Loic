Pour avoir la musique romantique en fond :

1. Télécharge une piste MP3 gratuite (romantique, piano, douce) sur :
   - https://pixabay.com/music/search/romantic/
   - ou YouTube Audio Library (musique libre de droits)

2. Enregistre le fichier ici sous le nom : romantic.mp3

Si le fichier n'existe pas, le bouton ♫ sera visible mais la musique ne jouera pas
(ajoute romantic.mp3 pour activer le son).
